Advanced information systems security

You can use network-based programs such as Snort and eTrust Intrusion Detection, which require installation on only one computer. The application (or service) scans all transmissions on a subnet to determine real-time network activity. This type of application acts both as a manager and as an agent. The network acts passively, and the host upon which the IDS (Intrusion Detection Systems) is installed does all the work. Network-based IDS applications are easy to establish and generally require the installation of only one program on one host. A network-based IDS is especially suited for thwarting scanning attacks and denial-of-service attacks.
